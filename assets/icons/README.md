# MKPE Icons and Logos

**Location**: `C:\mkpe\assets\icons\`

---

## Files

### SVG Logos (Vector, Scalable)

**mkpe_logo.svg** (Static)
- Hexagonal shield with concentric rings
- Cyan/blue color scheme
- Use for: Print, high-res displays, documentation

**mkpe_animated.svg** (Animated) ⭐ PRIMARY LOGO
- Same design as static version
- Features:
  - Breathing core glow (2.5s cycle)
  - Expanding protection rings (3s cycle, staggered)
  - Translucent blue emanations
- Use for: Web pages, app splash screens, presentations
- Opens in: Any modern browser, WebView

### ICO Files (Windows Icons)

**mkpe_app.ico** (32 KB)
- Application icon
- Used for: Installed applications, .exe files

**mkpe_tray.ico** (32 KB)
- System tray icon
- Used for: Running applications in taskbar

**mkpe_logo.ico** (32 KB)
- General purpose icon
- Converted from static SVG

---

## Usage

### For Web/HTML
```html
<img src="mkpe_animated.svg" alt="MKPE Logo" width="256" height="256">
```

### For Applications
- Windows: Use .ico files
- macOS/Linux: Convert SVG to .icns or .png

### For Documentation
- Use SVG for scalability
- Animated version for presentations
